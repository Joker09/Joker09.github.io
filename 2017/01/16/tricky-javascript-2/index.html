将函数表达式赋值给变量后无法通过函数名调用函数？

2017-01-16
kkk
JavaScript
函数表达试

var a = function b(){};
上面的函数可以通过a()来调用，但不能通过b()来调用。
解释： 在此作用域内(以window为例)，函数b只是a的一个值，并没有在全局进行函数声明。此写法实质上还是函数表达式，b也没能够作为window的一个属性，自然就无法通过window.b()来调用了

更新

b可以在函数内部调用，来实现如递归之类的操作。
