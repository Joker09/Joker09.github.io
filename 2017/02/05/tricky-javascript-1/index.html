toString() 报错

2017-02-05
kkk
JavaScript
toString

1.toString() 报错

解释器尝试将1.作为一个小数的前半部分。所以写成以下三种方式可避免这个问题。

1.0.toString();

(1).toString();

1..toString();

{}.toString() 报错

这次虽然没有小数的问题，但却存在另外一个问题。解释器将{作为一个块的开始，并没有把{}作为一个对象来看待。所以改写成({}).toString()即可。
