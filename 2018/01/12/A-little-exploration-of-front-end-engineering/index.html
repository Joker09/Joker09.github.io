数据的渲染方式

数据的渲染方式主要分为两种：前端渲染和后端渲染

  1. 前端渲染

    前端发送 ajax 请求，后台返回 json 数据，前台再用 json 数据拼接成字符串插入到 html 中。

  2. 后端渲染

    后端使用 ThinkPHP 模板引擎渲染。

二者的优缺点对比：

  • 采用前端渲染：后台只需关心数据，无需理会 html 的结构，也不用担心套模板文件时出错；缺点是需要提前确定好 json 数据结构，并且前端在开发时需要采用 MOCK 生成假数据。(其实还有一个跨域问题也不算大问题，只要后端同学设置 CORS 就行了)
